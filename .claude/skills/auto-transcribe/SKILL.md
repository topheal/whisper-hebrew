# סריקה ותמלול אוטומטי לפי דרישה

מריץ סריקה חד-פעמית של כל התיקיות שמוגדרות ב-`config.yaml`, לפי סדר העדיפות שם.

## הוראות

1. הרץ:
```
python "<תיקיית_הפרויקט>/auto_transcribe.py"
```

2. הצג את הפלט בזמן אמת — לכל קובץ שמטופל יודפס: התחלת תמלול, נתיב ה-SRT שנשמר, ואז סיכום.

3. בסיום — סכם למשתמש כמה קבצים טופלו ואיפה הפלט נשמר (`<שם>.srt` + `<שם>_summary.txt` באותה תיקייה של קובץ המקור).

## הערות
- קבצים עם שם זהה אך סיומת שונה (אודיו+וידאו) מתומללים פעם אחת בלבד
- קבצים שכבר תומללו (יש להם .srt + summary.txt עדכניים) מדולגים אוטומטית
- קבצים שעדיין מסתנכרנים (Google Drive) מדולגים ויטופלו בסריקה הבאה
- הסיכום האוטומטי מופק ע"י Claude Code headless (`claude -p`) - דרך המנוי הקיים, בלי מפתח API נפרד
- לשינוי תיקיות/תדירות/מקביליות — ערוך את config.yaml

# תמלול עברית עם Whisper

השתמש בכלי Bash כדי להריץ את סקריפט התמלול המקומי.

## הוראות

1. אם המשתמש סיפק נתיב לקובץ — הרץ:
```
python "<תיקיית_הפרויקט>/transcribe.py" "<נתיב_הקובץ>"
```

2. אם ההרצה נכשלת עם `ModuleNotFoundError` (חבילה כמו faster_whisper חסרה) — התקן את התלויות ונסה שוב:
```
pip install -r "<תיקיית_הפרויקט>/requirements.txt"
```
ואם זה לא עוזר, ודא ש-ffmpeg מותקן ונמצא ב-PATH (`ffmpeg -version`); אם לא, הסבר למשתמש איך להתקין (ראה README.md בפרויקט).

3. אם לא סיפק נתיב — בקש ממנו לציין נתיב מלא לקובץ.

4. אחרי הרצה מוצלחת — הצג את תוכן קובץ ה-.txt שנוצר.

## הערות
- בפעם הראשונה יורד מודל (~1.5GB) - זה עשוי לקחת כמה דקות
- הפלט נשמר אוטומטית ב-.txt ליד קובץ המקור
- לשינוי מודל: ערוך את MODEL_SIZE ב-transcribe.py
- ffmpeg הוא תלות חוץ-Python ולא מותקן אוטומטית ע"י pip - אם הוא חסר יש להפנות את המשתמש להוראות ההתקנה ב-README.md

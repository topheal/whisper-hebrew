# תמלול עברית עם Whisper

השתמש בכלי Bash כדי להריץ את סקריפט התמלול המקומי.

## הוראות

1. אם המשתמש סיפק נתיב לקובץ — הרץ מיד:
```
python "<תיקיית_הפרויקט>/transcribe.py" "<נתיב_הקובץ>"
```

2. אם לא סיפק נתיב — בקש ממנו לציין נתיב מלא לקובץ.

3. אחרי הרצה מוצלחת — הצג את תוכן קובץ ה-.txt שנוצר.

## הערות
- בפעם הראשונה יורד מודל (~1.5GB) - זה עשוי לקחת כמה דקות
- הפלט נשמר אוטומטית ב-.txt ליד קובץ המקור
- לשינוי מודל: ערוך את MODEL_SIZE ב-transcribe.py
